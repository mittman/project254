# project254
============

##### Build:
`$ make`

##### Usage:
*Write log_parsed.txt to current directory*  
`$ ./parser [filename]`

*Write to another location*  
`$ ./parser [filename] [output]`

*Print to terminal*  
`$ ./parser [filename] -`
